Usando ems

O em é uma unidade relativa no CSS. Seu valor é baseado no tamanho da fonte do elemento pai. Isso permite escalabilidade e melhor acessibilidade.

Por exemplo, se o elemento pai tem font-size de 16px, então 1em equivale a 16px, e 2em equivale a 32px.

Usando Percentual

A unidade % é baseada no tamanho do elemento pai. Por exemplo, se uma div tem largura de 500px, e uma outra div dentro dela tem width: 50%, essa segunda div terá 250px de largura.

Isso permite layouts mais flexíveis e adaptáveis a diferentes tamanhos de tela.

Usando Pixels

O px é uma unidade absoluta no CSS. Ele não muda conforme o tamanho da tela ou fonte, garantindo um tamanho fixo para os elementos.

Usar pixels pode ser útil quando você deseja um design fixo e previsível, mas pode ser menos responsivo.
